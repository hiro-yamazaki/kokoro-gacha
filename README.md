# こころのガチャ（レベルアップ版）

回すたびに、少しだけ気持ちが軽くなる。日常の小さな休憩のためのガチャアプリ。

## 課題内容（どんな作品か）

- 「じゃんけんアプリ」をベースに、クリック→ランダム抽選→画面表示 の構造を応用したガチャアプリ
- ボタンを押すと「ほっとする言葉」「小さなアドバイス」「短い格言」「自分を整える一言」が表示される
- **レアリティシステム（N / R / SR / SSR）** を搭載した重み付き抽選
- **コレクション機能**で取得済みメッセージを `localStorage` に保存し、コンプリート率を可視化
- レア度に応じた派手な演出（揺れの強化・カプセルのオーラ・全画面キラキラ）

## 工夫した点・こだわった点

- **レアリティ重み付き抽選**：`Math.random()` を応用し、SSR 3% / SR 12% / R 25% / N 60% の確率で抽選 → そのレア度内からランダムに1個選ぶ 2段階方式
- **localStorage によるコレクション機能**：取得済みのメッセージIDを配列で保存。ブラウザを閉じても進捗が残り、12種コンプリートでリセットボタンが出現する
- **レア度別の派手な演出**
  - N：通常の揺れ
  - R：緑のオーラ + 強めの揺れ（0.7秒）
  - SR：紫のオーラ + さらに強い揺れ（1.0秒）+ 紫グロー付き結果カード
  - SSR：金色オーラ + 揺れに拡縮を加えた最強アニメ（1.4秒）+ 全画面に金色フラッシュ + 12個の ✨ が画面下から舞い上がる演出
- **NEW! バッジ**：初めて出たメッセージには赤いバッジが一瞬光る
- **コンプリート時演出**：プログレスバーが金色シマーに変化、コレクション表示に ✨ コンプリート！ が付く
- **多重クリック防止**：アニメーション中はガチャボタンを無効化（SSR時はオーバーレイ終了まで）
- **テーマとの整合性**：自己肯定感が最も上がる「今日は、ちゃんとここまで来ただけで十分です。」を SSR に設定し、ご褒美感を演出
- **モダンなUI**：余白広めの縦長レイアウト・Noto Sans JP・立体感のある影、スマホでも見やすいmax-width 420px

## 難しかった点・次回トライしたいこと

- **アニメーションの再生制御**：同じクラスを連続で付与してもアニメーションが再生されない問題を、`void element.offsetWidth` での強制再描画で解決した
- **複数アニメーションの重ね合わせ**：SSR結果カードに `cardShimmer`（無限ループ）と `popIn`（一回再生）の両方を同時適用するときの記法（カンマ区切り）の理解が必要だった
- **次回トライしたいこと**
  - 効果音の追加（Web Audio API か `<audio>` 要素）
  - カプセル開封時の光線エフェクト（SVG マスクで実装）
  - PWA 化してホーム画面に追加できるようにする
  - メッセージのカテゴリ別フィルター機能

## 動作確認方法

1. `index.html` をブラウザで開く（または上記デプロイURL）
2. 「ガチャを回す」ボタンをクリック
3. レア度に応じた演出と共にメッセージが表示される
4. 「コレクション」表示が増えていき、12種類すべて引くとリセットボタンが出現
5. ブラウザを閉じて再度開いても進捗は保存されている

## レアリティ仕様

| レア度 | 確率 | 件数 | 演出 |
|---|---|---|---|
| SSR | 3% | 1 | 金色オーラ + 全画面キラキラ + 揺れ1.4秒 |
| SR | 12% | 2 | 紫オーラ + 揺れ1.0秒 + 結果カードに紫グロー |
| R | 25% | 4 | 緑オーラ + 揺れ0.7秒 |
| N | 60% | 5 | 通常の揺れ0.5秒 |
