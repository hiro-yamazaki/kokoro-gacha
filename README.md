# こころのガチャ v3（習慣化フォーカス版）

毎日3分、自分の心と向き合う習慣のためのガチャアプリ。
ガチャの楽しさで継続を担保し、メッセージで自己受容を促す。

## 課題内容（どんな作品か）

- 「じゃんけんアプリ」をベースに、クリック→ランダム抽選→画面表示 の構造を発展させた**習慣化型ガチャアプリ**
- **40種類のメッセージ**（カテゴリ4種、レアリティ4段階、気分・時間帯タグ付き）
- **デイリーメッセージ**（1日1回、日付シードで全員同じ結果＝「今日のあなたへ」感）
- **連続日数 (Streak)** 表示で毎日開きたくなる動機づけ
- **気分セレクト**（不安/疲れ/穏やか/燃えてる）で確率が変動
- **天井 (Pity) システム**（20回SR以上なし→確定、80回SSRなし→確定）
- **図鑑・マイカード・シェア**（取得済み一覧、お気に入り保存、画像化）
- **サウンド**（Web Audio API でシンセ合成、外部音源不要）

## 工夫した点・こだわった点

### ゲームデザインの工夫

1. **コアループの設計** — 「気分選ぶ → 引く → お気に入り保存 → 図鑑確認 → 翌日また開く」を1〜3分で1周完結に
2. **デイリーメッセージ** — 日付からハッシュを計算し、その日のメッセージを決定論的に固定。同じ日に何度開いても同じ結果が出ることで「今日のあなたへ」という特別感を演出
3. **連続日数システム** — サンクコスト効果で「途切れたくない」心理を生み、習慣化を後押し
4. **気分セレクト** — プレイヤーの選択を確率に反映（マッチで重み+2、時間帯マッチで+1）。運の要素を残しつつ「選んだ意味」を持たせる
5. **天井システム** — 20回でSR以上確定、80回でSSR確定。「いつかは当たる」安心感を担保しつつ、SR以上の希少性は維持
6. **確定演出** — SR/SSR時にガチャ揺れの前に紫/金のグローが一瞬光ることで、結果が出る前の「来るかも」期待感を作る
7. **NEW! 通知ドット** — 図鑑タブに赤い点が付き、未確認の新規取得を視覚的に伝える
8. **トースト通知** — 「新しいカードを獲得！」「今日の言葉が届きました」などのフィードバックで達成感を即時提供

### UI/UX の工夫

9. **モバイルファースト** — max-width 480px、ボトムナビゲーション、safe-area対応で本物のアプリ感
10. **タブ切替（今日 / 図鑑 / マイカード）** — 機能を3画面に分離し、迷わず操作できる
11. **サウンドデザイン** — タップ音、開封音、レア度別ファンファーレを Web Audio API で合成（外部ファイル不要、軽量、オフライン動作）
12. **設定モーダル** — サウンドON/OFF、進捗確認、天井までの残り回数、リセット
13. **シェア機能** — Canvas API でメッセージを画像化、ダウンロード可能。テキストコピーにも対応
14. **アクセシビリティ** — `aria-label` / `aria-pressed` / `aria-hidden` を設定、ESCキーでモーダル閉じる
15. **データマイグレーション** — v2 のローカルデータを v3 構造に自動移行する処理を実装

### 技術的なチャレンジ

16. **Web Audio API のシンセ合成** — `OscillatorNode` で波形を生成、`GainNode` でエンベロープを作って効果音を実装
17. **重み付きランダム抽選の2段階方式** — レア度を先に決定 → そのレア度内で重み付きランダム選択
18. **日付シード乱数** — 文字列ハッシュから決定論的に乱数を生成し、デイリーを実現
19. **localStorage の構造化保存** — 取得日時、お気に入り、ストリーク、天井カウンタを別キーで管理

## 動作確認方法

1. `index.html` をブラウザで開く（または上記デプロイURL）
2. 「今日のひとこと」カードをタップ → 今日のデイリーメッセージが届く
3. 気分を選択 → ガチャを回す
4. 結果を ♡ でお気に入り保存、📤 でシェア画像を生成
5. ボトムナビから「図鑑」「マイカード」へ
6. ブラウザを閉じても進捗・連続日数・お気に入りが保存される
7. 設定（⚙）からサウンドのON/OFF、リセット可能

## ゲームバランス（仕様）

| R | 25% | 12件 | 35% |
| N | 60% | 20件 | 40% |

| パラメータ | 値 |
|---|---|
| 天井（SR以上確定） | 20回 |
| 天井（SSR確定） | 80回 |

| 時間帯マッチによる重み | +1 |
